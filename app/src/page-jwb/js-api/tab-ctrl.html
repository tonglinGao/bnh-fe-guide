教务宝 tab-ctrl 组件详解

该组件是教务宝项目最重要的组件，实现的功能是点击左侧导航页面，会在该组件内实现以tab切换的形式展现，方便进行页面切换并实现记录用户的当前所有的已打开的页面，在初始化的时候一并加载。
