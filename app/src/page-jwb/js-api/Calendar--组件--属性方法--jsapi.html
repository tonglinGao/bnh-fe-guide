基础属性

todayClass - 今天样式对应的 class 名
类型：String 默认：cur_day
说明： 今天样式对应的 class 名
weekEnd - 周末样式对应的 class 名
类型：String 默认：week_end
说明： 周末样式对应的 class 名，六日的样式是统一的。没有做区分
call - 切换上下月按钮的回调函数
类型：Function 默认：无 ， 需要自己传值
说明： 切换上下月按钮的回调函数，有时候需要监听上下月按钮的事件可以通过此来实现。在函数在回到当前年月的时候不执行事件。上下月按钮没有作区分，有待改进。
yearBox - 盛放年的元素的id值
类型：String 默认：无 ， 需要自己传值
说明： 盛放年的元素对应的id值
monthBox - 盛放月的元素的id值
类型：String 默认：无 ， 需要自己传值
说明： 盛放月的元素对应的id值
calendarBox - 盛放日历主体的元素的id值
类型：String 默认：无 ， 需要自己传值
说明： 盛放日历主体的元素对应的id值
prevBtn - 上一月按钮元素的id值
类型：String 默认：无 ， 需要自己传值
说明： 上一月按钮元素对应的id值
nextBtn - 下一月按钮元素的id值
类型：String 默认：无 ， 需要自己传值
说明： 下一月按钮元素对应的id值

内置方法

init() - 初始化日历
参　数：无
说　明： 依据当前年月日期初始化日历可以做回到今天功能

示例

示例一：

// Calendar示例
// ==生成日历
var calendar = new Calendar({
    todayClass: "cur_day",
    weekEnd: "week_end",
    yearBox: "year",
    monthBox: "month",
    calendarBox: "calendar_box",
    prevBtn: "prev_btn",
    nextBtn: "next_btn",
    call: function(){
        alert("你点击了上月或者下月按钮");
    }
});
// ==回到今天
$("#go_today").click(function() {
    calendar.init();
});
